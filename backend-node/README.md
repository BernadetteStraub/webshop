# Webshop backend

## Installation


### Install dependencies

```npm install ```

### Run the server

```npm run dev```

### Run the tests

```npm run test```

### Run the linter

```npm run lint```

### Run the linter and fix errors

```npm run lint:fix```

### Upgrade the DB schema
```npx prisma generate```

```prisma migrate dev --name <MIGRATION_NAME>```
